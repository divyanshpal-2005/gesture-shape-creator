# hand gestures-shape-creator
A hand gestures-based shape drawing application that allows users to create and manipulate shapes using intuitive hand gestures.

Features
✋ Gesture-Based Controls
🖐 Open Hand → Spawn shapes
✊ Fist → Grab & throw objects
☝️ Point → Free draw mode
✌️ Peace → Clear canvas
🎯 Real-Time Hand Tracking
Uses MediaPipe for accurate hand detection
Smooth and responsive interaction
🔷 Multiple Shapes
Sphere, Cube, Cone, Star, Diamond, Hexagon, Ring, Cross
🎨 Customization
Adjustable shape size
Multiple color options + custom color picker
Separate draw color and shape color
🖌️ Free Drawing Mode
Draw in air using index finger
Undo strokes
⚡ Physics-Based Interaction
Gravity and motion effects
Throw objects naturally
🎮 Keyboard Shortcuts
Ctrl + Z → Undo
C → Clear
← / → → Change shapes
H → Toggle panels
🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling & UI effects
JavaScript (Vanilla JS) – Logic & interactions
MediaPipe Hands – Hand tracking & gesture recognition
📷 How It Works
Allow camera access
Show your hand in front of the webcam
Perform gestures to:
Create shapes
Move them
Draw freely

#Make sure to allow camera permissions

 Use Cases
Touchless UI systems
Interactive learning tools
Gesture-based gaming/drawing apps
Computer vision projects
